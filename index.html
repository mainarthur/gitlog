Naris

Информационная платформа для саморазвития, которая помогает прокачать навыки планирования, задать интересующие вопросы и получить на них ответы. Платформа позволяет создавать как публичные, так и закрытые (платные) материалы.

Технологии:
  • Typescript
  • Angular
  • NestJS
  • SQLite
Web | GitHub

OBSOverlay

Приложение используется в OBS Studio через плагин "Браузер". На сцену накладывается прозрачный слой, содержащий информационную графику, созданную с помощью HTML.

Управление осуществляется по принципу удаленного вызова процедур. Автор использует сторонний браузер для выбора информации и метода ее размещения на экране, после чего формируется управляющее сообщение, которое формирует точно такую же картинку на странице открытой в OBS Studio (этап синхронизации).

Технологии:
  • Javascript
  • Metarhia
GitHub

Elex

Словарная оболочка с веб-интерфейсом. Создание, редактирование, конвертация словарей происходит отдельно (через командную строку). Файлы загружаются по SCP/SFTP на сервер в специальную папку. Это программа для частного пользования.

Разыскиваются желающие улучшить программу. В частности на данный момент хотелось бы добавить routing для навигации по истории поиска (https://github.com/geniot/elex/issues/1).

Технологии:
  • Java
  • Spring Boot
  • Angular
Web | GitHub

NtscQT

Инструмент для стилизации фото и видео под старую VHS плёнку. Основан на алгоритме, максимально достоверно имитирующем аналоговый сигнал и выглядит аутентичнее существующих инструментов

Развиваю в свободное время, а в первую очередь хочется заняться оптимизацией и более гибкой настройкой

Технологии:
  • Python
  • PyQT
  • NumPy
  • FFmpeg
